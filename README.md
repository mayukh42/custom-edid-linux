# custom-edid-linux
Steps to add Custom EDID on Linux for AMD/ Intel GPUs. 

This is often required for very old monitors or TVs which these vendors send wrong color range signal to, resulting in washed out or darkened colors that are inaccurate. 

The guide can also be used to replace a different EDID for the device, to fix improper listing of resolution and refresh rate combinations.

Refer ```edid.md``` for details. It is written assuming Arch Linux or its variants.
